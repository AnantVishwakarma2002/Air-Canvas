======================================================================================
*******************************WELCOME TO AI AIR CANVAS*******************************

Requirements:
1. WebCam atleast 720Pixels at 60 fps
2. Python

Installing Various Python Libraries:
To install the python libraries needed for the successful execution of the code run the following commands in the command prompt of your pc:
pip install opencv-python
pip install numpy
pip install time
pip install mediapipe

Now run the Air_canvas.py file 



======================================================================================

DATA SET LINKS:
For performing the handtracking, position of the finger tips, and which fingers are up at the moment we have created a Track_hands_module.py file which is imported as Track_hands_Module in the Air_canvas.py file.
The Track_hands_module uses mediapipe library at the backend that comes filled with various methods to detect objects in the Video or the picture
